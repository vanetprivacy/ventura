What do ya want to know about?

 
  • Who made this?
    • This website was made as part of a project for a Client Side Webprogramming course offered by Saginaw Valley State University by Dustyn James Tubbs, a student of Computer Science from 2013-2017 at SVSU.
     
  • How was this made?
    • Dustyn developed a javascript decerator that coupled with another javascript library called pixijs in order to display the trace data. As for the data itself, we sourced from the Generic Mobility Simulation Framework's website: http://gmsf.sourceforge.net/
     
  • That's neat, can I try using it?
    • Yes! This code is available under the MIT free software license copyright 2014
     
  • So where is the code?
    • Sure, come fork our API's repo Right here
     
  • You're code is awful.
    • Not a question, but help us make it better!
     
  • I don't know how to use javascript but want to visualize some data, what do?
    • As of right now, just stay patient. There are plans in the future (written as of 9/16/2014) for expanding this site to take input trace data to visualize, so stay tuned!
